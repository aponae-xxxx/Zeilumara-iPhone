## [1.0.0] - 2026-02-01

### Added
- Initial release of Zeilumara Time for iPhone
- Dual clock display showing both Zeilumara and human time
- Real-time clock updates with live synchronization
- Event creation and management with Zeilumara time
- Local notification scheduling for events
- Automatic time conversion between Zeilumara and human time
- Timezone translation to multiple human timezones
- Event repeat support (both Zeilumara and human frequencies)
- Settings panel with customizable options:
  - Configurable epoch date
  - Display language (Chinese/Romanized/Both)
  - Theme selection (Light/Dark/Auto)
  - Notification preferences
- Data persistence with JSON storage
- Export/import functionality for events and settings
- Goddess awakening messages at special moments
- Comprehensive unit tests (>85% coverage)
- SwiftUI interface with accessibility support
- Local-first privacy (all data stored on device)

## Version History

### Version Naming Convention

- **Major.Minor.Patch** (e.g., 1.0.0)
  - **Major**: Breaking changes or major new features
  - **Minor**: New features, backward compatible
  - **Patch**: Bug fixes and small improvements

### Release Channels

- **Production**: Available on App Store
- **Beta**: Available via TestFlight
- **Alpha**: Internal testing only
